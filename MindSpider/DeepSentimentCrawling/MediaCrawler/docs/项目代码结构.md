# Project code structure

```
MediaCrawler
├── base
│ └── base_crawler.py # Abstract base class of the project
├── cache
│ ├── abs_cache.py # Cache abstract base class
│ ├── cache_factory.py # Cache factory
│ ├── local_cache.py # Local cache implementation
│ └── redis_cache.py # Redis cache implementation
├── cmd_arg
│ └── arg.py # Command line parameter definition
├── config
│ ├── base_config.py # Basic configuration
│ ├── db_config.py # Database configuration
│ └── ... # Configuration files for each platform
├── constant
│ └── ... # Definition of constants for each platform
├── database
│ ├── db.py # Database ORM, encapsulates addition, deletion, modification and query
│ ├── db_session.py # Database session management
│ └── models.py # Database model definition
├── docs
│ └── ... # Project documentation
├── libs
│ ├── douyin.js # Douyin Sign function
│ ├── stealth.min.js # JS that removes browser automation features
│ └── zhihu.js # Zhihu Sign function
├── media_platform
│ ├── bilibili # B station collection implementation
│ ├── douyin # Douyin collection implementation
│ ├── kuaishou # Kuaishou collection implementation
│ ├── tieba # Baidu Tieba collection implementation
│ ├── weibo # Weibo collection implementation
│ ├── xhs # Xiaohongshu collection implementation
│ └── zhihu # Zhihu collection implementation
├── model
│ ├── m_baidu_tieba.py # Baidu Tieba data model
│ ├── m_douyin.py # Douyin data model
│ ├── m_kuaishou.py # Kuaishou data model
│ ├── m_weibo.py # Weibo data model
│ ├── m_xiaohongshu.py # Xiaohongshu data model
│ └── m_zhihu.py # Zhihu data model
├── proxy
│ ├── base_proxy.py # Proxy base class
│ ├── providers # Agent provider implementation
│ ├── proxy_ip_pool.py # Proxy IP pool
│ └── types.py # Agent type definition
├── store
│ ├── bilibili # B station data storage implementation
│ ├── douyin # Douyin data storage implementation
│ ├── kuaishou # Kuaishou data storage implementation
│ ├── tieba # Tieba data storage implementation
│ ├── weibo # Weibo data storage implementation
│ ├── xhs # Xiaohongshu data storage implementation
│ └── zhihu # Zhihu data storage implementation
├── test
│ ├── test_db_sync.py # Database synchronization test
│ ├── test_proxy_ip_pool.py # Proxy IP pool test
│ └── ... # Other test cases
├── tools
│ ├── browser_launcher.py # Browser launcher
│ ├── cdp_browser.py # CDP browser control
│ ├── crawler_util.py # Crawler tool function
│ ├── utils.py # General utility function
│   └── ...
├── main.py # Program entry, supports --init_db parameter to initialize the database
├── recv_sms.py # SMS forwarding HTTP SERVER interface
└── var.py # Global context variable definition
```
