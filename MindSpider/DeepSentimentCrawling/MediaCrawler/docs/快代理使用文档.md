An example of initializing a fast proxy, as shown in the following code, requires 4 parameters

```python
# File address: proxy/providers/kuai_daili_proxy.py
# -*- coding: utf-8 -*-
def new_kuai_daili_proxy() -> KuaiDaiLiProxy:
    """
Construct a fast proxy HTTP instance
    Returns:

    """
    return KuaiDaiLiProxy(
kdl_secret_id=os.getenv("kdl_secret_id", "your fast proxy secert_id"),
kdl_signature=os.getenv("kdl_signature", "Your fast proxy signature"),
kdl_user_name=os.getenv("kdl_user_name", "your fast agent username"),
kdl_user_pwd=os.getenv("kdl_user_pwd", "your fast agent password"),
    )

```
